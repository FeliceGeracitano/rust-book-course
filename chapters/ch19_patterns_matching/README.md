# Chapter 19 — Patterns and Matching

> Book: https://doc.rust-lang.org/book/ch19-00-patterns.html

## Summary
The pattern-matching system in depth — where patterns appear, refutability, and
full pattern syntax.

## You will learn
- Patterns in `let`, `match`, `if let`, and function parameters
- Refutable vs. irrefutable patterns
- Ranges, bindings, guards, and destructuring

## Subchapters
- 19.1 All the Places Patterns Can Be Used
- 19.2 Refutability: Whether a Pattern Might Fail to Match
- 19.3 Pattern Syntax
