# Chapter 7 — Packages, Crates, and Modules

> Book: https://doc.rust-lang.org/book/ch07-00-managing-growing-projects-with-packages-crates-and-modules.html

## Summary
Organize larger projects with packages, crates, modules, paths, and `use`.

## You will learn
- Package vs. crate
- The module tree and `pub` visibility
- Absolute and relative paths
- The `use` keyword
- Splitting modules across files

## Subchapters
- 7.1 Packages and Crates
- 7.2 Control Scope and Privacy with Modules
- 7.3 Paths for Referring to an Item in the Module Tree
- 7.4 Bringing Paths Into Scope with the `use` Keyword
- 7.5 Separating Modules into Different Files
