# Chapter 1 — Getting Started

> Book: https://doc.rust-lang.org/book/ch01-00-getting-started.html

## Summary
Install Rust with `rustup`, compile a single file with `rustc`, then meet
**Cargo** — Rust's build system and package manager that you'll use for the rest
of the course.

## You will learn
- Install and update the toolchain with `rustup`
- Compile and run a one-file program with `rustc`
- Create, build, run, and check a project with `cargo`

## Subchapters
- 1.1 Installation
- 1.2 Hello, World!
- 1.3 Hello, Cargo!
