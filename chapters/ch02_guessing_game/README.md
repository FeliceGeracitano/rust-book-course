# Chapter 2 — Programming a Guessing Game

> Book: https://doc.rust-lang.org/book/ch02-00-guessing-game-tutorial.html

## Summary
Build a number-guessing game end to end — your first real program tying together
input, variables, types, branching, and looping.

## You will learn
- Read input from stdin
- Parse strings into numbers
- Use external crates (`rand`)
- `match` on comparisons
- Loop until the guess is correct
