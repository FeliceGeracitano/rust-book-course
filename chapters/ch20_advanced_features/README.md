# Chapter 20 — Advanced Features

> Book: https://doc.rust-lang.org/book/ch20-00-advanced-features.html

## Summary
The sharp tools — unsafe Rust, advanced traits and types, advanced
functions/closures, and macros.

## You will learn
- `unsafe` Rust
- Associated types and operator overloading
- Newtype pattern and type aliases
- Function pointers and returning closures
- Declarative and procedural macros

## Subchapters
- 20.1 Unsafe Rust
- 20.2 Advanced Traits
- 20.3 Advanced Types
- 20.4 Advanced Functions and Closures
- 20.5 Macros
