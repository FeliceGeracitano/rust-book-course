# Chapter 10 — Generic Types, Traits, and Lifetimes

> Book: https://doc.rust-lang.org/book/ch10-00-generics.html

## Summary
Write code over many types with generics, define shared behavior with traits,
and keep references valid with lifetimes.

## You will learn
- Generic functions, structs, and enums
- Trait definitions and trait bounds
- Lifetime annotations and why they exist

## Subchapters
- 10.1 Generic Data Types
- 10.2 Defining Shared Behavior with Traits
- 10.3 Validating References with Lifetimes
