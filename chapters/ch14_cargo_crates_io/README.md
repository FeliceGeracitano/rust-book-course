# Chapter 14 — More about Cargo and Crates.io

> Book: https://doc.rust-lang.org/book/ch14-00-more-about-cargo.html

## Summary
Go deeper on Cargo — release profiles, publishing, workspaces, installs, and
custom commands.

## You will learn
- Release profiles
- Publishing to crates.io and writing docs
- Multi-crate workspaces
- `cargo install`
- Extending Cargo with custom subcommands

## Subchapters
- 14.1 Customizing Builds with Release Profiles
- 14.2 Publishing a Crate to Crates.io
- 14.3 Cargo Workspaces
- 14.4 Installing Binaries with `cargo install`
- 14.5 Extending Cargo with Custom Commands
