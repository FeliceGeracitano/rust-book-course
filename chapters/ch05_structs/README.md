# Chapter 5 — Using Structs to Structure Related Data

> Book: https://doc.rust-lang.org/book/ch05-00-structs.html

## Summary
Model related data with structs, then add behavior with methods.

## You will learn
- Define and instantiate structs
- Field init shorthand and struct update syntax
- Tuple structs
- `impl` blocks: methods vs. associated functions

## Subchapters
- 5.1 Defining and Instantiating Structs
- 5.2 An Example Program Using Structs
- 5.3 Method Syntax
