# Chapter 11 — Writing Automated Tests

> Book: https://doc.rust-lang.org/book/ch11-00-testing.html

## Summary
Write, run, and organize automated tests in Rust.

## You will learn
- `#[test]`, `assert!`, and `assert_eq!`
- `should_panic` and testing for failure
- Filtering and controlling test runs
- Unit vs. integration tests

## Subchapters
- 11.1 How to Write Tests
- 11.2 Controlling How Tests Are Run
- 11.3 Test Organization

## Exercises
Open `src/lib.rs`, complete each `todo!()`, then make the tests pass:

```bash
cargo test -p ch11_testing
```
