# Chapter 6 — Enums and Pattern Matching

> Book: https://doc.rust-lang.org/book/ch06-00-enums.html

## Summary
Enums express "one of several" values; `match` and `if let` destructure them
exhaustively.

## You will learn
- Define enums and their variants
- `Option<T>` and why Rust has no null
- `match` arms and value binding
- `if let` and `let...else`

## Subchapters
- 6.1 Defining an Enum
- 6.2 The `match` Control Flow Construct
- 6.3 Concise Control Flow with `if let` and `let...else`
