# Chapter 3 — Common Programming Concepts

> Book: https://doc.rust-lang.org/book/ch03-00-common-programming-concepts.html

## Summary
The universal building blocks — variables and mutability, scalar and compound
types, functions, comments, and control flow.

## You will learn
- `let`, `mut`, `const`, and shadowing
- Scalar types (integer, float, bool, char) and compound types (tuple, array)
- Function parameters and return values
- Control flow with `if`, `loop`, `while`, and `for`

## Subchapters
- 3.1 Variables and Mutability
- 3.2 Data Types
- 3.3 Functions
- 3.4 Comments
- 3.5 Control Flow
